# GreenTech Solutions Website

## Project Description

GreenTech Solutions is a responsive and accessible website designed for a fictional environmental sustainability company. The website promotes eco-friendly technology products and provides educational resources about sustainability and green innovation.

The project was created as a final web development project using HTML5, CSS3, and JavaScript.

---

## Website Features

### Homepage
- Company introduction
- Mission statement
- Featured eco-friendly technology products
- Call-to-action button

### About Us Page
- Company story and mission
- Timeline of company achievements
- Team information

### Products & Services Page
- Product showcase cards
- Product search/filter functionality
- Product images and descriptions

### Blog & Resources Page
- Sustainability articles
- Educational resources
- Eco-friendly technology tips

### Contact Page
- Contact form with validation
- Email, phone number, and address
- Accessible form labels

### Final Page
- Summary of project features and technologies used
